Política de Privacidade — SyntaxVET

Última atualização: junho de 2026

A SyntaxVET ("nós", "nosso") desenvolveu o aplicativo SyntaxVET como um serviço comercial. Esta página informa sobre nossas políticas de coleta, uso e divulgação de informações pessoais para usuários do serviço.

1. Informações coletadas

Coletamos as seguintes informações para a prestação do serviço:

  • Dados de cadastro: nome, e-mail, número de CRMV e imagem de assinatura digital do veterinário.
  • Dados de pacientes: nome do animal, espécie, raça, idade e nome do tutor, fornecidos pelo veterinário.
  • Gravações de voz: áudios gravados pelo veterinário durante o uso do app, utilizados exclusivamente para transcrição e geração de laudos.
  • Dados de pagamento: processados pelo Stripe. Não armazenamos dados de cartão de crédito.

2. Uso das informações

As informações coletadas são utilizadas para:

  • Geração de laudos veterinários em formato PDF
  • Envio de laudos via WhatsApp
  • Processamento de pagamentos e gerenciamento de assinaturas
  • Melhoria contínua do serviço

3. Compartilhamento de dados

Não vendemos dados pessoais. Compartilhamos dados apenas com prestadores de serviço essenciais:

  • Supabase: armazenamento seguro de dados
  • OpenAI (Whisper): transcrição de áudio
  • Anthropic (Claude): estruturação de laudos
  • Stripe: processamento de pagamentos

4. Retenção de dados

Os dados são mantidos enquanto a conta estiver ativa. O veterinário pode solicitar a exclusão de seus dados a qualquer momento pelo e-mail contato@syntaxvet.com.

5. Segurança

Utilizamos medidas técnicas e organizacionais para proteger seus dados, incluindo criptografia em trânsito e em repouso.

6. LGPD

Em conformidade com a Lei Geral de Proteção de Dados (Lei nº 13.709/2018), você tem direito a acessar, corrigir e solicitar a exclusão de seus dados pessoais. Entre em contato pelo e-mail contato@syntaxvet.com.

7. Contato

Dúvidas sobre esta política: contato@syntaxvet.com
